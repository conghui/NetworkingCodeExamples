# Tracker example

This is an Maxeler DFE networking example project. Please see the included text document for further details as it includes useful illustrations.

## Running

```
$ source ../../config.sh
<...>
$ cd bitstream
$ maxJavaRun TrackerManager
<...>
$ cd ../runtime
$ cp <path-to-new-max-file> .
$ ./build.py
<...>
$ ./run.py
<...>
```
